```

                ↓
2. 📦 โอน: Transfer → คลัง
                ↓
       ┌────────┴────────┐
       ▼                 ▼
3a. ✅ ขายให้      3b. 🚨 พบปัญหา
    ลูกค้า                ↓
                  🔧 ส่งกลับกรอ
                          ↓
                  4. 🔁 กรอ: เริ่มกรอ → ชั่งใหม่ (Lot ใหม่)
                          ↓
                  ┌───────┴───────┐
                  ▼               ▼
```

## 📍 จุดสำคัญ (Key Rules)

1. **Lot No อัตโนมัติ**: `YY(ปี) + Machine + CustCode(4) + MM(เดือน)`
   - ตัวอย่าง: `69 + BL01 + 0001 + 05` = `69BL01000105`

2. **WO > SO > Lot** ลำดับชั้นใช้ทั้งระบบ (Dashboard, History, Transfer, Rework)

3. **ม้วนกรอ + เศษเสีย → บังคับกรอกเหตุผล** (เพื่อ Pareto analysis)

4. **Roll No อุดเลขที่ลบ** ป้องกัน gap

5. **2 ระบบกรอ**: จากเป่า (auto) vs นอกระบบ (manual)

6. **3 ประเภทโอน**: FG (สีน้ำเงิน) / กรอ (ส้ม) / เศษ (แดง) — ใบโอนแยกชัดเจน
