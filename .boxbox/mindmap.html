4
🖨️ พิมพ์ Label อัตโนมัติ
หลังบันทึกม้วน ระบบพิมพ์ label ให้อัตโนมัติ พร้อม Barcode + QR Code
📄
ใบยาว (165×70 mm)
แสดงข้อมูลครบ: สินค้า, ลูกค้า, Mat Code, น้ำหนัก, QR Code ออกแบบ layout ได้เองใน Label Designer
🏷️
ใบสั้น (76×76 mm)
เหมาะกับเครื่องพิมพ์ thermal ขนาดเล็ก ข้อมูลสรุปย่อ + QR Code
🔲
QR Code
เมื่อสแกน QR จะเปิดหน้ารายละเอียดม้วนนั้นทันที — ใช้ตรวจสอบสินค้าได้ทุกที่
✏️
Label Designer
ออกแบบตำแหน่ง field บน label ได้เอง ลาก-วาง ปรับขนาดตัวอักษร ตั้งค่าหัวกระดาษ
5
📦 โอนเข้าคลัง & จัดการคลังสินค้า

📤
โอนเข้าคลัง
เลือกม้วนที่ต้องการโอน กดยืนยัน สถานะเปลี่ยนเป็น "โอนแล้ว" ทันที

✅ ยืนยันการโอนแบบ batch
🔄 อัปเดต Realtime ทุกหน้าจอ
🏭
คลังสินค้า

🔍 ค้นหาม้วนด้วย Barcode/QR
📊 สรุปยอดตามสินค้า
📄 Export รายงาน
6
📊 รายงาน & Dashboard
ติดตามผลผลิต วิเคราะห์ข้อมูล ย้อนดูประวัติ
📊
Dashboard ภาพรวม
  • กราฟผลผลิตรายวัน/รายเดือน
  • ยอดรวมแต่ละแผนก
  • % Yield (ของดี vs เสีย)
  • เปรียบเทียบช่วงเวลา
📋
รายงาน SO / WO
  • ผลผลิตตาม Sale Order
  • สถานะ Work Order
  • ยอดส่งมอบ vs เป้า
  • รายงานลูกค้า
📅
ประวัติการผลิต
  • ย้อนดูม้วนทุกใบ
  • ลบม้วน + บันทึกเหตุผล
  • ค้นหาตาม Lot/เครื่อง
  • ดาวน์โหลด log
🔧
รายงานกรอ (Rework)
  • ม้วนที่ต้องกรอใหม่
  • สาเหตุและเหตุผล
  • ยอดเศษเสียแต่ละประเภท
  • Inbox งานรอกรอ
🔐
Admin Panel
ต้องกรอก PIN พิเศษเพื่อเข้าถึง — สำหรับผู้ดูแลระบบเท่านั้น
🔐
ผู้ดูแลระบบ (Admin)
จัดการการตั้งค่าระดับสูง ดูแล PIN แต่ละแผนก ตรวจสอบ log การลบข้อมูล และจัดการระบบทั้งหมด
🔑 จัดการ PIN แผนก 📋 ดู Deletion Logs ⚙️ ตั้งค่าระบบ 👤 สิทธิ์ผู้ใช้

☁️ ฐานข้อมูล Supabase (Cloud)
