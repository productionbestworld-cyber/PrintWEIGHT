# Flow ระบบชั่งม้วนงานพิมพ์

## เมนูหลัก

ระบบตอนนี้เหลือใช้งานหลัก 4 หน้า

1. **ตั้งค่าเครื่องชั่ง**
   - ตั้งค่าเครื่องชั่ง / เครื่องสลิท / ข้อมูลงานที่ผูกกับเครื่อง
2. **ตั้งงาน**
   - สร้างงานผลิตจาก SO / WO / Lot
   - เลือกลูกค้าและสินค้าเป็นดรอปดาวน์
   - ระบบช่วยสร้าง Lot ให้อัตโนมัติ
3. **ชั่ง**
   - เลือกเครื่องพิมพ์
   - เลือกเครื่องสลิท
   - ชั่งม้วนตามขั้นตอน
4. **โอน**
   - โอนรายการม้วนที่ชั่งเสร็จแล้ว

## Flow การทำงานจริง

1. เข้าเว็บ
2. กด **เข้าใช้งานแผนกพิมพ์**
3. ใส่ PIN
4. ไปที่หน้า **ตั้งงาน**
5. กรอก SO
6. กรอก WO
7. เลือกเครื่องสลิท
8. เลือกเครื่องพิมพ์
9. เลือกลูกค้า
10. เลือกสินค้า
11. ระบบสร้าง Lot ให้เอง
12. กดบันทึกงาน
13. ไปหน้า **ชั่ง**
14. เลือกเครื่องพิมพ์
15. เลือกเครื่องสลิท
16. ชั่งม้วนใสที่รับมา ว่ามีกี่กิโล
17. นำม้วนไปพิมพ์
18. ชั่งเศษเสียจากงานพิมพ์ เช่น เศษสี / เศษติดกาว
19. ชั่งม้วนใหญ่ที่พิมพ์ได้
20. นำม้วนใหญ่ไปสลิท
21. ชั่งเศษข้างสลิท
22. ชั่งม้วนสำเร็จรูปที่ส่งลูกค้า
23. ปริ้นลาเบล
24. ไปหน้า **โอน** ถ้าต้องโอนงานออก

## ลำดับการชั่ง

1. **ม้วนก่อนพิมพ์**
   - เป็นม้วนใสก่อนนำไปพิมพ์
   - ใช้บันทึกว่าอินพุตรับมากี่กิโล
2. **เศษจากพิมพ์**
   - เศษสี
   - เศษติดกาว
3. **ม้วนใหญ่หลังพิมพ์**
   - ม้วนที่พิมพ์เสร็จแล้ว ก่อนนำไปสลิท
4. **เศษข้างสลิท**
   - เศษขอบ / เศษข้างที่เสียตอนซอยม้วน
5. **ม้วนสลิทส่งลูกค้า**
   - ม้วนเล็กหลังซอย
   - ใช้ปริ้นลาเบลส่งลูกค้า

## กติกา SO / WO / Lot

- 1 SO มีหลาย WO ได้
- 1 WO ต้องเริ่มม้วนที่ 1 ได้ชุดเดียว
- งานทุกอย่างต้องผูกกับ Lot
- Lot ควรให้ระบบเจน ไม่ควรให้พนักงานพิมพ์เอง

## สูตร Lot

ตัวอย่าง Lot:

```text
69SL1P1TC00107
```

ความหมาย:

```text
69    = ปี พ.ศ. 2569 เอา 2 ตัวท้าย
SL1   = เครื่องสลิท
P1    = เครื่องพิมพ์
TC001 = รหัสสินค้า
07    = เดือน
```

รูปแบบเต็ม:

```text
ปี + เครื่องสลิท + เครื่องพิมพ์ + รหัสสินค้า + เดือน
```

ตัวอย่าง:

```text
69SL1P1TC00107
69SL2P3GP02507
69SL4P2NP01108
```

## จำนวนเครื่อง

เครื่องพิมพ์มี 3 เครื่อง

```text
P1
P2
P3
```

เครื่องสลิทมี 4 เครื่อง

```text
SL1
SL2
SL3
SL4
```

## วิธีตั้งงาน

หน้า **ตั้งงาน** ให้ทำตามนี้

1. ใส่ SO
2. ใส่ WO
3. เลือกเครื่องสลิท เช่น SL1
4. เลือกเครื่องพิมพ์ เช่น P1
5. เลือกลูกค้า
6. เลือกสินค้า
7. ระบบจะสร้าง Lot ให้ เช่น `69SL1P1TC00107`
8. ตรวจข้อมูล
9. กดบันทึกงาน

ถ้า Lot ยังไม่ขึ้น ให้เช็คว่าเลือกครบหรือยัง

- เครื่องสลิท
- เครื่องพิมพ์
- สินค้า / รหัสสินค้า

## SQL ที่ต้องมีใน production_jobs

ถ้าฐานข้อมูลยังไม่มีช่องเครื่องพิมพ์และเครื่องสลิท ให้รัน SQL นี้

```sql
ALTER TABLE production_jobs ADD COLUMN IF NOT EXISTS print_machine TEXT;
ALTER TABLE production_jobs ADD COLUMN IF NOT EXISTS slit_machine TEXT;

CREATE INDEX IF NOT EXISTS idx_production_jobs_print_machine
  ON production_jobs(print_machine);
CREATE INDEX IF NOT EXISTS idx_production_jobs_slit_machine
  ON production_jobs(slit_machine);
```
