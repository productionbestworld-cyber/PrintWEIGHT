# Flow ระบบชั่งม้วนงานพิมพ์

## เมนูหลัก

1. **ตั้งงาน**
   - ใส่ SO / WO
   - เลือกลูกค้าและสินค้า
   - เลือกเครื่องพิมพ์ 3 เครื่อง: P1, P2, P3
   - เลือกเครื่องสลิท 4 เครื่อง: SL1, SL2, SL3, SL4
   - ระบบสร้าง Lot ให้อัตโนมัติ

2. **ชั่ง**
   - เลือกงานที่ตั้งไว้
   - ชั่งตามลำดับจริงของงาน
   - แยกประเภทงานชั่งด้วย `inbound_type`

3. **โอน**
   - ใช้หลังชั่งม้วนสำเร็จแล้ว
   - โอนเฉพาะม้วนสำเร็จส่งลูกค้า

4. **ตั้งค่า**
   - ต้องใส่รหัส Admin
   - เปิด/ปิดเฟคจำนวนกิโล
   - เปิด/ปิดการพิมพ์ใบปะหน้าอัตโนมัติ

## ลำดับการชั่งใน 1 งาน

| ลำดับ | ประเภท | ความหมาย |
|---:|---|---|
| 1 | ม้วนก่อนพิมพ์ | ชั่งม้วนวัตถุดิบก่อนนำไปพิมพ์ |
| 2 | เศษพิมพ์ / เศษสี | เศษเสียจากขั้นตอนพิมพ์ |
| 3 | เศษกาว | เศษเสียจากกาว |
| 4 | ม้วนใหญ่หลังพิมพ์ | ม้วนใหญ่ที่ออกจากเครื่องพิมพ์ |
| 5 | เศษข้างสลิท | เศษขอบ/เศษข้างตอนซอยม้วน |
| 6 | ม้วนสลิทส่งลูกค้า | ม้วนสำเร็จหลังสลิท ใช้ปริ้นลาเบลและโอน |

## สรุปตอนปิดงาน

ตอนกด **ปิดงาน** ระบบจะแสดงและพิมพ์สรุปแยกเป็น:

| รายการ | วิธีคิด |
|---|---|
| ม้วนก่อนพิมพ์ | รวม kg ของ `inbound_type = input_roll` |
| เศษพิมพ์ / เศษสี | รวม kg ของ `roll_type = scrap_print_color` |
| เศษกาว | รวม kg ของ `roll_type = scrap_glue` |
| ม้วนใหญ่ที่พิมพ์ได้ | รวม kg ของ `inbound_type = printed_jumbo` |
| เศษข้างสลิท | รวม kg ของ `roll_type = scrap_slit_side` |
| ม้วนสำเร็จหลังสลิท | รวม kg ของ `inbound_type = slit_roll` |
| เศษรวม | เศษสี + เศษกาว + เศษข้างสลิท |
| Yield | ม้วนสำเร็จหลังสลิท ÷ ม้วนก่อนพิมพ์ |
