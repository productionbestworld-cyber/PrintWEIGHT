🌬 แผนกเป่า (BLOW)

คู่มือการใช้งานระบบชั่งน้ำหนักม้วน

บริษัท เบสท์เวิลด์ อินเตอร์พลาส จำกัด · BWP ProductionWeight
ภาพรวมงานแผนกเป่า: ชั่งม้วนที่ผลิตได้ → แยกเป็น ม้วนดี / ม้วนกรอ / เศษ → โอนม้วนดีเข้าคลัง · ส่งม้วนเสียไปกรอ
🌬 ชั่งม้วน→ ✅ ม้วนดี → คลัง → ส่งลูกค้า
→ 🔧 ม้วนกรอ → ส่งกรอ/รอพิจารณา→แผนกกรอ
→ 🗑 เศษ → รวมยอดเศษ

1 เข้าระบบ & เลือกแผนก

  1. เปิดเว็บระบบ → จะเจอหน้า เลือกแผนก
  2. กดเลือก 🌬 ผลิต (เป่า)
  3. ใส่ PIN ของแผนก → กดยืนยัน
  4. มุมซ้ายบนจะแสดง 🌬 ผลิต(เป่า) = อยู่แผนกเป่าถูกต้อง
ถ้าเลือกแผนกผิด กดที่ชื่อแผนกมุมซ้ายบน → เลือกใหม่ได้ตลอด

2 เลือกงาน / เครื่อง (ก่อนชั่ง)

  1. ไปเมนู ⚖ ชั่งน้ำหนัก
  2. เลือก เครื่อง ที่กำลังเดิน (เช่น BL01)
  3. เลือกงาน (Lot / Sales Order) — หรือกด 📂 ดึงงานเก่า ถ้าทำงานเดิมต่อ
  4. ตรวจชื่อสินค้า / ลูกค้า / ขนาด ให้ตรงกับที่ผลิตจริง

3 ชั่งม้วนดี (FG)

  1. เลือกประเภท ม้วนดี
  2. วางม้วนบนตาชั่ง → รอตัวเลข นิ่ง (stable)
  3. ใส่ชื่อ ผู้ตรวจ (ครั้งแรกของกะ)
  4. กด บันทึก → ระบบให้เลขม้วนอัตโนมัติ + พิมพ์ป้าย
เลขม้วนเริ่มที่ #1 ไล่ขึ้นเอง · ถ้าเลขชนระบบหาเลขใหม่ให้อัตโนมัติ ไม่ต้องกังวล

4 ชั่งม้วนกรอ (ม้วนเสีย/ต้องแก้)

เลือกประเภท ม้วนกรอ แล้ว เลือก 1 ใน 2 แบบ:

เลือกใช้เมื่อม้วนจะไป...
✓ ส่งกรอ มั่นใจว่ากรอแก้ได้ เข้าคิวแผนกกรอ (ผ่านการโอน)
รอพิจารณา ไม่แน่ใจว่ากรอได้ไหม ส่งให้ ผจก ตัดสิน

ใส่ เหตุผล ที่ม้วนเสีย (จำเป็น) → กด บันทึก

สำคัญ: ระบุเหตุผลทุกครั้ง — ใช้ดูสาเหตุเสียที่ Dashboard เพื่อแก้ปัญหาที่ต้นทาง

5 ชั่งเศษ

  1. เลือกประเภท เศษ → เลือกชนิด: เศษใส / เศษสี / เศษก้อน
  2. ชั่งทั้งถุง (ระบบใช้น้ำหนักเต็ม ไม่หักแกน) → ใส่เหตุผล → บันทึก

6 โอนม้วนดี + ม้วนกรอ เข้าคลัง

  1. ไปเมนู 📦 โอนเข้าคลัง
  2. เลือกม้วนที่จะโอน (ม้วนดี + ม้วนกรอที่กด "ส่งกรอ")
  3. กดยืนยันโอน → ม้วนดีเข้าสต็อกคลัง · ม้วนกรอเข้าคิวแผนกกรอ
ห้ามลืมขั้นนี้! ถ้าไม่โอน ม้วนกรอ "ส่งกรอ" จะค้าง ไม่ไปถึงแผนกกรอ และจะขึ้นเป็น "ยังไม่จัดการ" ใน Dashboard

7 คลังสินค้า & แจ้ง NC

  1. เมนู 🏬 คลังสินค้า → แท็บ สต็อกคงเหลือ ดูม้วนดีที่พร้อมส่ง
  2. กดที่ม้วน → ดูรายละเอียด → ถ้าเจอม้วนเสียในคลัง กด ⚠ แจ้ง NC
  3. แท็บ เศษ = ดูยอดเศษทั้งหมด · แท็บ จัดส่ง = ออกใบส่งของให้ลูกค้า
ม้วนที่แจ้ง NC จะมี แถบแดงเตือน + ขีดคร่อมในตาราง บอกชัดว่าม้วนไหนถูกแจ้งออกไป

8 ดูภาพรวม (Dashboard)

  • เมนู 📊 Dashboard → 🎛 ศูนย์ควบคุม เห็นครบในหน้าเดียว
  • การ์ด ผลิตรวม / FG(ดี) / ม้วนกรอ / เศษรวม + แตกรายละเอียดได้
  • การ์ด 🔧 ม้วนกรอ → กดบรรทัด "▼ ยังไม่จัดการ" เพื่อเช็คม้วนตกค้างที่ต้องเคลียร์

9 ปัญหาที่พบบ่อย

อาการทำยังไง
ม้วนกรอไม่ไปถึงแผนกกรอเช็คว่า โอนเข้าคลัง แล้วหรือยัง (ข้อ 6)
ตัวเลขชั่งไม่นิ่ง บันทึกไม่ได้รอจนตัวเลขนิ่ง (stable) ก่อนกดบันทึก
เน็ตหลุดตอนชั่งระบบเก็บคิวออฟไลน์ให้ พอเน็ตกลับมาจะส่งเอง
มี badge แดงบนเมนูมีม้วนรอพิจารณา/NC → กดเข้าไปเคลียร์
เลือกแผนกผิดกดชื่อแผนกมุมซ้ายบน → เปลี่ยน
📌 จำ 3 ข้อหลัก: ① ชั่งให้ตรงประเภท (ดี/กรอ/เศษ)  ② ใส่เหตุผลม้วนกรอทุกครั้ง  ③ อย่าลืมโอนเข้าคลัง
