## 1. ภาพรวมระบบ

ระบบนี้ใช้บันทึกการชั่งน้ำหนักม้วนพลาสติกของทุกแผนก แล้วเก็บเข้าฐานข้อมูลกลาง พร้อมพิมพ์ใบปะหน้า และสรุปรายงานให้อัตโนมัติ

**3 ส่วนที่ทำงานร่วมกัน:**

| ส่วน | คืออะไร | อยู่ที่ไหน |
|---|---|---|
| **เว็บแอป** | หน้าชั่ง / รายงาน / ข้อมูล | เบราว์เซอร์ (Chrome/Edge) |
| **ตัวต่อเครื่องชั่ง (Bridge)** | อ่านค่าน้ำหนักจากเครื่องชั่ง → ส่งเข้าเว็บ | ไอคอน "BWP Scale" บนหน้าจอ |
| **เครื่องชั่งจริง** | ฮาร์ดแวร์ | ต่อสาย COM |

**ประเภทม้วนที่ชั่งได้:**
- **ม้วนดี (FG)** — ม้วนที่ผ่าน ส่งขายได้
- **ม้วนกรอ** — ม้วนเสียที่ยังกรอแก้ได้ → ส่งแผนกกรอ
- **เศษเสีย** — เศษใส / เศษสี / เศษก้อน (กรอไม่ได้ ทิ้ง)

## 2. เริ่มต้นใช้งาน

### 2.1 เลือกแผนก
เปิดเว็บครั้งแรก → เจอหน้า **"เลือกแผนกเพื่อเข้าสู่ระบบ"** → กดเลือกแผนกตัวเอง:
- 🌬 **ผลิต(เป่า)**
- 🖨 **ผลิต(พิมพ์)**
- 🔁 **กรอ(Rework)**

### 2.2 ใส่ PIN แผนก
ใส่รหัส PIN ของแผนก (ค่าเริ่มต้น):
- เป่า = `1111` · พิมพ์ = `2222` · กรอ = `3333`

> 🔄 **สลับแผนก**: กดชื่อแผนกมุมซ้ายบน → เลือกแผนกใหม่ → ใส่ PIN ของแผนกนั้น

### 2.3 เมนูหลัก (แถบบน)
| เมนู | ใช้ทำอะไร | ล็อก? |
|---|---|---|
| ชั่งน้ำหนัก | หน้าหลัก — ชั่งม้วน | — |
| โอนเข้าคลัง | โอนม้วนดีเข้าคลัง | — |
| คลังสินค้า | ดูสต็อกในคลัง | — |
| Dashboard | ภาพรวม/รายงาน | — |
| ประวัติผลิต | งานที่ปิดแล้ว | — |
| 📦 คลังข้อมูล | เพิ่ม/แก้ ลูกค้า+สินค้า | — |
| 🔒 พิจารณาม้วนกรอ | ผจก ตัดสินม้วนเสีย | PIN ผจก |
| 🔒 NC (คลัง/QC) | ผจก ตัดสินของเสีย | PIN ผจก |
| ตั้งค่าเครื่อง | ตั้งโปรไฟล์เครื่อง | — |

## 3. เครื่องชั่ง (Bridge)

### 3.1 เปิดเครื่องชั่ง
- ปกติ Bridge **เปิดเองอัตโนมัติ** ทุกครั้งที่เปิดคอม (ทำงานเบื้องหลัง)
- ถ้าค่าน้ำหนักไม่ขึ้น → ดับเบิลคลิกไอคอน **"BWP Scale"** บนหน้าจอ → กด **▶ เปิดเครื่องชั่ง**

### 3.2 ตั้งค่า COM port (ทำครั้งเดียว)
1. ดับเบิลคลิกไอคอน **BWP Scale**
2. กด **⚙ ตั้งค่า COM port**
3. เลือก COM port ของเครื่องชั่ง (มีชื่ออุปกรณ์บอก) + ความเร็ว (ปกติ 9600)
4. กด **✓ บันทึก**

### 3.3 แชร์เครื่องชั่งกับโปรแกรมเก่า
ถ้าใช้โปรแกรมเก่าคู่กัน — ที่ `http://localhost:8080`:
- **⏸ ปล่อยให้โปรแกรมเก่า** → เปิดโปรแกรมเก่าใช้ได้
- ปิดโปรแกรมเก่า → Bridge จับเครื่องชั่งกลับเอง (หรือกด **▶ เอากลับมา**)

## 4. แผนกผลิต (เป่า / พิมพ์)

> ทั้งเป่าและพิมพ์ใช้หน้าชั่งเหมือนกัน

### 4.1 ตั้งงาน (ก่อนชั่งม้วนแรก)
ที่หน้า **ชั่งน้ำหนัก** กดการ์ดเครื่อง → กรอกข้อมูลงาน:

1. **เครื่อง** — เลือกเครื่องที่ผลิต
2. **Item Code** — พิมพ์ค้นหา → เลือก → ระบบ**เติมให้อัตโนมัติ**: ชื่อสินค้า, ขนาด, ลูกค้า, Mat Code, น้ำหนักแกน
   - 💡 ถ้าพิมพ์ **Mat Code** ที่ตรงกับสินค้า ก็เด้งข้อมูลให้เหมือนกัน
   - 💡 ถ้า Item Code **ยังไม่มีในระบบ** → กรอกรายละเอียด → กดปุ่มเขียว **"➕ บันทึกเป็นสินค้าใหม่"**
3. **Mat Code / น้ำหนักแกน** — เด้งให้อัตโนมัติ (ถ้าไม่มีให้กรอกเอง)
4. **Lot No** — คลิกช่อง → ระบบสร้างให้อัตโนมัติ (หรือพิมพ์เอง)
5. **ผู้ตรวจสอบ** — ชื่อคนที่ตรวจ (สำคัญ! ใช้แยกกะการทำงาน)
6. กด **✓ บันทึก + พร้อมใช้งาน**

### 4.2 ชั่งม้วน
1. วางม้วนบนเครื่องชั่ง → รอตัวเลข **"นิ่ง"** (ขึ้น ✓)
2. เลือกประเภทม้วน:
   - **ม้วนดี** (ปกติ)
   - **ม้วนกรอ** — ม้วนเสียที่กรอแก้ได้
   - **เศษเสีย** — เลือกชนิด: เศษใส / เศษสี / เศษก้อน
3. กด **บันทึก** → ระบบหักน้ำหนักแกนให้ (สุทธิ = รวม − แกน)
4. **ยกม้วนออก** ก่อนชั่งม้วนถัดไป (ระบบกันชั่งซ้ำ)
5. ใบปะหน้าพิมพ์อัตโนมัติ (ม้วนดี = ใบปกติ, เศษ = ใบปะเศษ)

> **ม้วนกรอ มี 2 แบบ:**
> - **กรอได้** (ปกติ) — ผลิตมั่นใจว่ากรอแก้ได้ → ส่งแผนกกรอเลย
> - **รอพิจารณา** — ไม่แน่ใจ → ส่งให้ ผจก ตัดสินก่อน (ไปโผล่ที่ "พิจารณาม้วนกรอ")

### 4.3 พักงาน / ปิดงาน
- **พักงาน** — เก็บงานไว้ชั่งต่อทีหลัง (ม้วนที่ชั่งแล้วยังอยู่)
- **ปิดงาน** — จบงานนี้ → ระบบสรุปยอด (ม้วนดี/กรอ/เศษ + Yield) + พิมพ์ใบสรุป

### 4.4 ขนาด/ใบปะหน้า
- ระบบจำขนาดใบปะหน้า (ใบยาว/ใบสั้น) ตามที่ตั้งใน Item Code
- เศษเสียจะพิมพ์ **ใบปะเศษ** (Waste Label) ขาว-ดำ อัตโนมัติ

## 5. แผนกกรอ (Rework)

แผนกกรอ = รับม้วนเสียจากผลิตมาแก้ (กรอใหม่) ให้กลายเป็นม้วนดี

### 5.1 ดูม้วนที่รอกรอ
เข้าหน้า **ชั่งน้ำหนัก** → แท็บ **🏭 รับจากผลิต**
- บนสุดมี **แบนเนอร์แจ้งเตือน** บอกว่ามีม้วนรอกี่ม้วน กี่กิโล กี่ใบสั่งผลิต
- ที่แท็บ "รับจากผลิต" มี **ตัวเลขสีแดง** บอกจำนวนม้วนรอ (อัปเดตเอง)

### 5.2 รับม้วนเข้ากรอ
รายการม้วนจัดกลุ่ม **ใบสั่งผลิต (WO) › SO › Lot** กดดูได้
แต่ละม้วนมี 3 ปุ่ม:
- **🔧 เริ่มกรอ** — รับเข้ากรอ (กรอกชื่อผู้รับ + วิธีกรอ) → สร้างงานกรอ
- **↩ ส่งคืนผลิต** — กรอไม่ได้ ส่งกลับให้ ผจก พิจารณาใหม่
- **🗑 เศษ** — กรอไม่ได้จริง → ทิ้งเป็นเศษ

> คอลัมน์ **"แหล่ง"** บอกที่มาของม้วน เช่น "งานจากแผนกเป่า", "ลูกค้าคืน" ฯลฯ

### 5.3 ชั่งม้วนที่กรอเสร็จ
1. แท็บ **🔁 งานกรอ (ชั่งน้ำหนัก)** → กดการ์ดงานที่จะกรอ
2. **เลือกเครื่องกรอ** (S01–S04) → ระบบสร้าง Lot ใหม่ให้
3. ชั่งม้วนที่กรอได้ (เหมือนแผนกผลิต) → ม้วนดีจากกรอจะนับเป็น FG
4. ปิดงานเมื่อกรอครบ

### 5.4 สร้างงานกรอเอง
แท็บงานกรอ → **+ สร้างงานใหม่** → เลือก Item Code (เด้งข้อมูลให้) → บันทึก

## 6. คลังข้อมูล

เมนู **📦 คลังข้อมูล** (พนักงานเข้าได้ ไม่ต้องใส่ PIN) — มี 2 แท็บ

### 6.1 แท็บ ลูกค้า
- ตารางลูกค้าทั้งหมด เรียงตามรหัส
- **+ เพิ่มลูกค้า** / คลิกแถวเพื่อแก้ไข
- **📥 Export Excel** — ออกเป็นไฟล์ (ลูกค้า + สินค้าของลูกค้า)

### 6.2 แท็บ Item Code (สินค้า)
ตารางสินค้าทั้งหมด แสดงครบ: Item Code · รหัสสินค้า · ชื่อ · ขนาด · **Mat Code** · **น้ำหนักแกน** · ลูกค้า
- **+ เพิ่ม Item** — เพิ่มสินค้าใหม่ (กรอกครบทุกช่อง)
- **✏ ดินสอ** — แก้ไข
- **⬆ นำเข้า Excel** — เพิ่มทีละเยอะจากไฟล์
- **📥 Export** — ออกเป็น Excel

> 💡 ระบบ "จำ" ค่า Mat Code / แกน ที่พนักงานกรอกตอนชั่ง → เติมเข้าคลังข้อมูลให้เอง ครั้งหน้าเลือกสินค้าก็เด้งให้

## 9. งานของผู้จัดการ

> 🔒 ล็อกด้วย **PIN ผจก** (ค่าเริ่มต้น `9999`)

### 9.1 พิจารณาม้วนกรอ
ม้วนที่ผลิตส่งมา "รอพิจารณา" → ผจก ตัดสิน:
- **กรอ** — ส่งแผนกกรอ
- **เก็บไว้** — เก็บเป็นสต็อก
- **เศษเสีย** — ทิ้ง

### 9.2 NC (คลัง/QC)
ของเสียที่คืนจากคลัง/ตรวจไม่ผ่าน → ผจก ตัดสินเหมือนกัน

## 10. Admin

> 🔒 PIN ผจก — เมนู Admin (มุมขวา)

- **ออกแบบใบปะหน้า** — แก้รูปแบบใบปะหน้า (ใบยาว/ใบสั้น/ใบเศษ) ลากวางได้ + ทดสอบปริ้น
- **เปลี่ยน PIN** — เปลี่ยนรหัส Admin + แต่ละแผนก
- **💾 Backup ทั้งระบบ** — ดาวน์โหลดทั้งฐานข้อมูลเป็น Excel (แนะนำทำทุกสัปดาห์)

## 11. แก้ปัญหาเบื้องต้น

| อาการ | วิธีแก้ |
|---|---|
| ค่าน้ำหนักไม่ขึ้น | เปิดไอคอน BWP Scale → ▶ เปิดเครื่องชั่ง → เช็ก COM port |
| ตัวเลขไม่นิ่ง / ชั่งไม่ได้ | รอจนขึ้น ✓ "นิ่ง" ก่อนกดบันทึก |
| เลือกสินค้าแล้วไม่เด้งข้อมูล | สินค้านั้นยังไม่มีในระบบ → เพิ่มที่ คลังข้อมูล หรือกดปุ่มเขียวตอนชั่ง |
| เข้าพิจารณาม้วนกรอไม่ได้ | ต้องใส่ PIN ผจก |
| เว็บไม่อัปเดต | กด Ctrl+Shift+R (รีเฟรชล้าง cache) |
| อยากใช้โปรแกรมเก่าคู่กัน | BWP Scale → ปล่อยให้โปรแกรมเก่า |
| กลัวข้อมูลหาย | Admin → 💾 Backup ทั้งระบบ (เก็บไฟล์ไว้) |
